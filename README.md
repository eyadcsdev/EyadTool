# Eyad Tools

أدوات مفتوحة المصدر لمشكلات عملية — مساحة يضع فيها إياد الحطامي الأدوات التي يطوّرها لتبسيط المهام المتكررة وتحويل المشكلات العملية إلى حلول سهلة الاستخدام.

---

## الأدوات المتاحة

### كشف جاهز

**للتعليم** — أداة مخصصة لمدرسي اللغة الإنجليزية في مركز اللغات والترجمة، تساعدهم على تجهيز كشف الدرجات النهائي من خلال إزالة أعمدة الحساب المؤقتة، وتحويل المعادلات إلى قيم ثابتة، ثم تنزيل نسخة جاهزة للتسليم.

يدعم الإصدار الحالي نموذج كشف الدرجات المعتمد في مركز اللغات والترجمة فقط.

---

## متطلبات التشغيل

- **PHP** 8.3 أو أحدث
- **Composer**
- **Node.js** 20+ و **npm**
- **SQLite** (للتطوير والتجارب — لا تعتمد الأدوات على قاعدة بيانات)
- امتدادات PHP المطلوبة من PhpSpreadsheet:
  - `ext-zip`
  - `ext-gd` (اختياري، لمعاينة الصور داخل ملفات Excel)
  - `ext-xml`

## الإعداد المحلّي

```bash
# استنساخ المستودع
git clone <repository-url>
cd eyadtool

# تثبيت اعتماديات PHP
composer install

# نسخ ملف البيئة وضبطه
cp .env.example .env
php artisan key:generate

# إنشاء قاعدة بيانات SQLite للتطوير
touch database/database.sqlite
php artisan migrate

# تثبيت اعتماديات الواجهة وبناءها
npm install
npm run build
```

## تشغيل خادم التطوير

```bash
composer run dev
```

يفتح المتصفح على `http://localhost:8000`.

## الخصوصية

- تُعالَج الملفات المرفوعة مؤقتًا فقط.
- لا تُحفظ الملفات في حسابات مستخدمين أو قاعدة بيانات.
- تُحذف النسخة المؤقتة بعد تجهيز ملف التنزيل.
- لا تُسجَّل أسماء الطلاب أو درجاتهم أو أي محتوى داخل ملفات Excel.
- لا يوجد نظام مستخدمين، ولا تسجيل دخول، ولا لوحة تحكم.
